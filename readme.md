# ToDoList

Класическое веб приложение "ToDoList" с использованием HTML, CSS, и JavaScript.

## Начало работы

  1. Клонируйте репозиторий на свой компьютер.
  2. Откройте index.html файл в браузере чтобы увидеть проект.
